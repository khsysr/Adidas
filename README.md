## 🙋 프로젝트 소개


<br>


- JSON 파일 사용: 아디다스 쇼핑몰의 제품 정보는 JSON 파일에 저장되어 있습니다.

- useEffect와 useState 사용: 리액트에서 useEffect와 useState를 사용하여 데이터 로딩 및 관리를 합니다.
